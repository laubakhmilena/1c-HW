# Домашнее задание №1 по 1С  
## Тема: Создание справочника «Сотрудники»

### Описание работы
В рамках данного задания была разработана конфигурация в системе **1С**, предназначенная для ведения учета сотрудников организации.  
Основной объект разработки — справочник **«Сотрудники»**, в котором реализовано хранение основной кадровой информации по каждому сотруднику.

### Цель задания
Создать справочник, позволяющий хранить и обрабатывать сведения о сотрудниках, а именно:
- табельный номер;
- ФИО;
- паспортные данные;
- дату рождения;
- адрес;
- образование;
- признак «Штатный».

### Реализованный функционал
В ходе выполнения задания были выполнены следующие действия:

1. **Создан справочник «Сотрудники»**  
   Справочник предназначен для хранения карточек сотрудников.

2. **Настроены стандартные реквизиты**  
   - в качестве табельного номера использован стандартный реквизит **Код**;
   - для удобства отображения ему был задан синоним **«Табельный номер»**;
   - для хранения ФИО используется стандартный реквизит, предназначенный для наименования элемента справочника.

3. **Настроены дополнительные реквизиты справочника**
   Были добавлены следующие поля:
   - Паспортные данные;
   - Дата рождения;
   - Адрес;
   - Образование;
   - Штатный.

4. **Реализован фиксированный список значений для реквизита «Образование»**  
   Пользователь может выбрать одно из заранее заданных значений:
   - высшее;
   - среднее;
   - среднее специальное.

5. **Добавлен логический реквизит «Штатный»**  
   Этот реквизит позволяет определить, является ли сотрудник штатным работником организации.

6. **Настроена форма элемента справочника**  
   Все добавленные реквизиты выведены в карточку сотрудника для удобного ввода и просмотра данных.

### Дополнительное задание
Для будущего улучшения предусмотрена реализация ввода реквизита **«Паспортные данные»** по шаблону, чтобы исключить ввод символов, отличных от цифр.  
Это позволит повысить корректность вводимой информации и сократить количество ошибок пользователя.
