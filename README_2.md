# Домашнее задание по 1С  
## Тема: Создание документа «Начисление зарплаты»

### Описание работы
В рамках данной работы в системе **1С:Предприятие** был разработан документ **«Начисление зарплаты»**, предназначенный для фиксации сведений о заработной плате сотрудников за месяц.  
Документ позволяет вводить данные по нескольким сотрудникам в одной форме, хранить дату ввода информации, а также отражать итоговую сумму начислений по документу.

### Цель работы
Реализовать в прикладном решении механизм учета начисления заработной платы сотрудников за месяц с возможностью:
- фиксировать дату ввода документа;
- вносить данные по сотрудникам в табличную часть;
- хранить ставку, количество отработанных часов и сумму;
- выводить общую сумму документа.

### Постановка задания
В документе необходимо было реализовать:
- дату ввода информации о заработной плате;
- табличную часть с полями:
  - **Сотрудник**;
  - **Ставка**;
  - **Количество часов**;
  - **Сумма**.
- дополнительное задание: реализовать вывод **общей суммы документа**.

> Примечание: по условию базового задания сумма по строке могла вводиться вручную.

### Выполненные этапы
В ходе работы были выполнены следующие действия:

1. **Создан документ «НачислениеЗарплаты»**  
   В конфигурации 1С был добавлен новый документ с синонимом **«Начисление зарплаты»**.

2. **Создана табличная часть «Сотрудники»**  
   Табличная часть используется для ввода данных по каждому сотруднику в рамках одного документа.

3. **Добавлены реквизиты табличной части**
   В табличную часть были включены следующие реквизиты:
   - `Сотрудник`
   - `Ставка`
   - `КоличествоЧасов`
   - `Сумма`

4. **Настроены типы данных**
   - реквизит `Сотрудник` связан со справочником **Сотрудники**;
   - `Ставка`, `КоличествоЧасов` и `Сумма` имеют числовой тип.

5. **Создана и настроена форма документа**
   В форме документа были выведены:
   - номер документа;
   - дата документа;
   - табличная часть с колонками:
     - Сотрудник;
     - Ставка;
     - Количество часов;
     - Сумма.

6. **Проверена работа документа в режиме 1С:Предприятие**
   Документ был успешно открыт в пользовательском режиме, после чего в него были внесены данные по нескольким сотрудникам.

7. **Реализован вывод общей суммы документа**
   Для выполнения дополнительного задания был создан реквизит документа:
   - `СуммаИтог`

   Этот реквизит был выведен на форму с заголовком **«Сумма (итог)»**.

8. **Добавлен код автоматического подсчета итоговой суммы**
   В модуле объекта документа была реализована процедура `ПередЗаписью`, которая суммирует значения поля `Сумма` по всем строкам табличной части и записывает результат в реквизит `СуммаИтог`.

### Использованный алгоритм расчета
При записи документа выполняется следующий принцип:
- цикл проходит по всем строкам табличной части **Сотрудники**;
- значения реквизита **Сумма** суммируются;
- итог записывается в реквизит **СуммаИтог**.

### Результат работы
В результате выполнения задания был создан рабочий документ **«Начисление зарплаты»**, который:
- фиксирует дату ввода;
- позволяет вносить данные по нескольким сотрудникам;
- хранит сведения о ставке, количестве часов и сумме;
- отображает общую сумму начислений по документу.

Поставленная задача была выполнена успешно.  
В конфигурации 1С реализован документ для учета начисления заработной платы сотрудников с табличной частью и выводом итоговой суммы по документу. Работа демонстрирует базовые навыки создания документов, табличных частей, реквизитов, форм и использования кода в модуле объекта для автоматизации расчетов.
